The root system of the exceptional Lie group E 8. Lie groups have many symmetries.

occurs not only in [geometry](https://en.wikipedia.org/wiki/Geometry "Geometry"), but also in other branches of mathematics. Symmetry is a type of [invariance](https://en.wikipedia.org/wiki/Invariant_\(mathematics\) "Invariant (mathematics)"): the property that a mathematical object remains unchanged under a set of [operations](https://en.wikipedia.org/wiki/Operation_\(mathematics\) "Operation (mathematics)") or [transformations](https://en.wikipedia.org/wiki/Transformation_\(mathematics\) "Transformation (mathematics)").[^1]

Given a structured object *X* of any sort, a [symmetry](https://en.wikipedia.org/wiki/Symmetry "Symmetry") is a [mapping](https://en.wikipedia.org/wiki/Mapping_\(mathematics\) "Mapping (mathematics)") of the object onto itself which preserves the structure. This can occur in many ways; for example, if *X* is a set with no additional structure, a symmetry is a [bijective](https://en.wikipedia.org/wiki/Bijection "Bijection") map from the set to itself, giving rise to [permutation groups](https://en.wikipedia.org/wiki/Permutation_group "Permutation group"). If the object *X* is a set of points in the plane with its [metric](https://en.wikipedia.org/wiki/Metric_\(mathematics\) "Metric (mathematics)") structure or any other [metric space](https://en.wikipedia.org/wiki/Metric_space "Metric space"), a symmetry is a [bijection](https://en.wikipedia.org/wiki/Bijection "Bijection") of the set to itself which preserves the distance between each pair of points (i.e., an [isometry](https://en.wikipedia.org/wiki/Isometry "Isometry")).

In general, every kind of structure in mathematics will have its own kind of symmetry, many of which are listed in the given points mentioned above.

## Symmetry in geometry

The types of symmetry considered in basic geometry include [reflectional symmetry](https://en.wikipedia.org/wiki/Reflection_symmetry "Reflection symmetry"), [rotational symmetry](https://en.wikipedia.org/wiki/Rotational_symmetry "Rotational symmetry"), [translational symmetry](https://en.wikipedia.org/wiki/Translational_symmetry "Translational symmetry") and [glide reflection symmetry](https://en.wikipedia.org/wiki/Glide_reflection_symmetry "Glide reflection symmetry"), which are described more fully in the main article [Symmetry (geometry)](https://en.wikipedia.org/wiki/Symmetry_\(geometry\) "Symmetry (geometry)").

## Symmetry in calculus

### Even and odd functions

#### Even functions

![](https://upload.wikimedia.org/wikipedia/commons/thumb/d/dd/Function_x%5E2.svg/250px-Function_x%5E2.svg.png)

ƒ ( x ) = 2 is an example of an even function.

Let *f* (*x*) be a [real](https://en.wikipedia.org/wiki/Real_number "Real number") -valued function of a real variable, then *f* is **even** if the following equation holds for all *x* and *\-x* in the domain of *f*:

${\displaystyle f(x)=f(-x)}$

Geometrically speaking, the graph face of an even function is [symmetric](https://en.wikipedia.org/wiki/Symmetry "Symmetry") with respect to the *y* -axis, meaning that its [graph](https://en.wikipedia.org/wiki/Graph_of_a_function "Graph of a function") remains unchanged after [reflection](https://en.wikipedia.org/wiki/Reflection_\(mathematics\) "Reflection (mathematics)") about the *y* -axis. Examples of even functions include [| *x* |](https://en.wikipedia.org/wiki/Absolute_value "Absolute value"), *x* <sup>2</sup>, *x* <sup>4</sup>, [cos](https://en.wikipedia.org/wiki/Trigonometric_function "Trigonometric function") (*x*), and [cosh](https://en.wikipedia.org/wiki/Hyperbolic_function "Hyperbolic function") (*x*).

#### Odd functions

![](https://upload.wikimedia.org/wikipedia/commons/thumb/f/f5/Function_x%5E3.svg/250px-Function_x%5E3.svg.png)

ƒ ( x ) = 3 is an example of an odd function.

Again, let *f* be a [real](https://en.wikipedia.org/wiki/Real_number "Real number") -valued function of a real variable, then *f* is **odd** if the following equation holds for all *x* and *\-x* in the domain of *f*:

${\displaystyle -f(x)=f(-x)}$

That is,

${\displaystyle f(x)+f(-x)=0\,.}$

Geometrically, the graph of an odd function has rotational symmetry with respect to the [origin](https://en.wikipedia.org/wiki/Origin_\(mathematics\) "Origin (mathematics)"), meaning that its [graph](https://en.wikipedia.org/wiki/Graph_of_a_function "Graph of a function") remains unchanged after [rotation](https://en.wikipedia.org/wiki/Coordinate_rotation "Coordinate rotation") of 180 [degrees](https://en.wikipedia.org/wiki/Degree_\(angle\) "Degree (angle)") about the origin. Examples of odd functions are *x*, *x* <sup>3</sup>, [sin](https://en.wikipedia.org/wiki/Sine "Sine") (*x*), [sinh](https://en.wikipedia.org/wiki/Hyperbolic_function "Hyperbolic function") (*x*), and [erf](https://en.wikipedia.org/wiki/Error_function "Error function") (*x*).

### Integrating

The [integral](https://en.wikipedia.org/wiki/Integral "Integral") of an odd function from − *A* to + *A* is zero, provided that *A* is finite and that the function is integrable (e.g., has no vertical asymptotes between − *A* and *A*).[^3]

The integral of an even function from − *A* to + *A* is twice the integral from 0 to + *A*, provided that *A* is finite and the function is integrable (e.g., has no vertical asymptotes between − *A* and *A*).[^3] This also holds true when *A* is infinite, but only if the integral converges.

### Series

- The [Maclaurin series](https://en.wikipedia.org/wiki/Maclaurin_series "Maclaurin series") of an even function includes only even powers.
- The Maclaurin series of an odd function includes only odd powers.
- The [Fourier series](https://en.wikipedia.org/wiki/Fourier_series "Fourier series") of a [periodic](https://en.wikipedia.org/wiki/Periodic_function "Periodic function") even function includes only [cosine](https://en.wikipedia.org/wiki/Trigonometric_function "Trigonometric function") terms.
- The Fourier series of a periodic odd function includes only [sine](https://en.wikipedia.org/wiki/Trigonometric_function "Trigonometric function") terms.

## Symmetry in linear algebra

### Symmetry in matrices

In [linear algebra](https://en.wikipedia.org/wiki/Linear_algebra "Linear algebra"), a **symmetric matrix** is a [square matrix](https://en.wikipedia.org/wiki/Square_matrix "Square matrix") that is equal to its [transpose](https://en.wikipedia.org/wiki/Transpose "Transpose") (i.e., it is invariant under matrix transposition). Formally, matrix *A* is symmetric if

${\displaystyle A=A^{T}.}$

By the definition of matrix equality, which requires that the entries in all corresponding positions be equal, equal matrices must have the same dimensions (as matrices of different sizes or shapes cannot be equal). Consequently, only square matrices can be symmetric.

The entries of a symmetric matrix are symmetric with respect to the [main diagonal](https://en.wikipedia.org/wiki/Main_diagonal "Main diagonal"). So if the entries are written as *A* = (*a* <sub><i>ij</i></sub>), then *a* <sub><i>ij</i></sub> = a <sub><i>ji</i></sub>, for all indices *i* and *j*.

For example, the following 3×3 matrix is symmetric:

${\displaystyle {\begin{bmatrix}1&7&3\\7&4&-5\\3&-5&6\end{bmatrix}}}$

Every square [diagonal matrix](https://en.wikipedia.org/wiki/Diagonal_matrix "Diagonal matrix") is symmetric, since all off-diagonal entries are zero. Similarly, each diagonal element of a [skew-symmetric matrix](https://en.wikipedia.org/wiki/Skew-symmetric_matrix "Skew-symmetric matrix") must be zero, since each is its own negative.

In linear algebra, a [real](https://en.wikipedia.org/wiki/Real_number "Real number") symmetric matrix represents a [self-adjoint operator](https://en.wikipedia.org/wiki/Self-adjoint_operator "Self-adjoint operator") over a [real](https://en.wikipedia.org/wiki/Real_number "Real number") [inner product space](https://en.wikipedia.org/wiki/Inner_product_space "Inner product space"). The corresponding object for a [complex](https://en.wikipedia.org/wiki/Complex_number "Complex number") inner product space is a [Hermitian matrix](https://en.wikipedia.org/wiki/Hermitian_matrix "Hermitian matrix") with complex-valued entries, which is equal to its [conjugate transpose](https://en.wikipedia.org/wiki/Conjugate_transpose "Conjugate transpose"). Therefore, in linear algebra over the complex numbers, it is often assumed that a symmetric matrix refers to one which has real-valued entries. Symmetric matrices appear naturally in a variety of applications, and typical numerical linear algebra software makes special accommodations for them.

## Symmetry in abstract algebra

### Symmetric groups

The **symmetric group** *S* <sub><i>n</i></sub> (on a [finite set](https://en.wikipedia.org/wiki/Finite_set "Finite set") of *n* symbols) is the [group](https://en.wikipedia.org/wiki/Group_\(mathematics\) "Group (mathematics)") whose elements are all the [permutations](https://en.wikipedia.org/wiki/Permutations "Permutations") of the *n* symbols, and whose [group operation](https://en.wikipedia.org/wiki/Group_operation "Group operation") is the [composition](https://en.wikipedia.org/wiki/Function_composition "Function composition") of such permutations, which are treated as [bijective functions](https://en.wikipedia.org/wiki/Bijection "Bijection") from the set of symbols to itself.[^4] Since there are *n*! (*n* [factorial](https://en.wikipedia.org/wiki/Factorial "Factorial")) possible permutations of a set of *n* symbols, it follows that the [order](https://en.wikipedia.org/wiki/Order_\(group_theory\) "Order (group theory)") (i.e., the number of elements) of the symmetric group *S* <sub><i>n</i></sub> is *n*!.

### Symmetric polynomials

A **symmetric polynomial** is a [polynomial](https://en.wikipedia.org/wiki/Polynomial "Polynomial") *P* (*X* <sub>1</sub>, *X* <sub>2</sub>,..., *X* <sub><i>n</i></sub>) in *n* variables, such that if any of the variables are interchanged, one obtains the same polynomial. Formally, *P* is a *symmetric polynomial* if for any [permutation](https://en.wikipedia.org/wiki/Permutation "Permutation") σ of the subscripts 1, 2,..., *n*, one has *P* (*X* <sub>σ(1)</sub>, *X* <sub>σ(2)</sub>,..., *X* <sub>σ(<i>n</i>)</sub>) = *P* (*X* <sub>1</sub>, *X* <sub>2</sub>,..., *X* <sub><i>n</i></sub>).

Symmetric polynomials arise naturally in the study of the relation between the roots of a polynomial in one variable and its coefficients, since the coefficients can be given by polynomial expressions in the roots, and all roots play a similar role in this setting. From this point of view, the [elementary symmetric polynomials](https://en.wikipedia.org/wiki/Elementary_symmetric_polynomial "Elementary symmetric polynomial") are the most fundamental symmetric polynomials. A [theorem](https://en.wikipedia.org/wiki/Elementary_symmetric_polynomial#The_fundamental_theorem_of_symmetric_polynomials "Elementary symmetric polynomial") states that any symmetric polynomial can be expressed in terms of elementary symmetric polynomials, which implies that every *symmetric* [polynomial expression](https://en.wikipedia.org/wiki/Polynomial_expression "Polynomial expression") in the roots of a [monic polynomial](https://en.wikipedia.org/wiki/Monic_polynomial "Monic polynomial") can alternatively be given as a polynomial expression in the coefficients of the polynomial.

#### Examples

In two variables *X* <sub>1</sub> and *X* <sub>2</sub>, one has symmetric polynomials such as:

- ${\displaystyle X_{1}^{3}+X_{2}^{3}-7}$
- ${\displaystyle 4X_{1}^{2}X_{2}^{2}+X_{1}^{3}X_{2}+X_{1}X_{2}^{3}+(X_{1}+X_{2})^{4}}$

and in three variables *X* <sub>1</sub>, *X* <sub>2</sub> and *X* <sub>3</sub>, one has as a symmetric polynomial:

- ${\displaystyle X_{1}X_{2}X_{3}-2X_{1}X_{2}-2X_{1}X_{3}-2X_{2}X_{3}\,}$

### Symmetric tensors

In [mathematics](https://en.wikipedia.org/wiki/Mathematics "Mathematics"), a **symmetric tensor** is [tensor](https://en.wikipedia.org/wiki/Tensor "Tensor") that is invariant under a [permutation](https://en.wikipedia.org/wiki/Permutation "Permutation") of its vector arguments:

${\displaystyle T(v_{1},v_{2},\dots ,v_{r})=T(v_{\sigma 1},v_{\sigma 2},\dots ,v_{\sigma r})}$

for every permutation σ of the symbols {1,2,...,*r* }. Alternatively, an *r* <sup>th</sup> order symmetric tensor represented in coordinates as a quantity with *r* indices satisfies

${\displaystyle T_{i_{1}i_{2}\dots i_{r}}=T_{i_{\sigma 1}i_{\sigma 2}\dots i_{\sigma r}}.}$

The space of symmetric tensors of rank *r* on a finite-dimensional [vector space](https://en.wikipedia.org/wiki/Vector_space "Vector space") is [naturally isomorphic](https://en.wikipedia.org/wiki/Natural_isomorphism "Natural isomorphism") to the dual of the space of [homogeneous polynomials](https://en.wikipedia.org/wiki/Homogeneous_polynomial "Homogeneous polynomial") of degree *r* on *V*. Over [fields](https://en.wikipedia.org/wiki/Field_\(mathematics\) "Field (mathematics)") of [characteristic zero](https://en.wikipedia.org/wiki/Characteristic_zero "Characteristic zero"), the [graded vector space](https://en.wikipedia.org/wiki/Graded_vector_space "Graded vector space") of all symmetric tensors can be naturally identified with the [symmetric algebra](https://en.wikipedia.org/wiki/Symmetric_algebra "Symmetric algebra") on *V*. A related concept is that of the [antisymmetric tensor](https://en.wikipedia.org/wiki/Antisymmetric_tensor "Antisymmetric tensor") or [alternating form](https://en.wikipedia.org/wiki/Alternating_form "Alternating form"). Symmetric tensors occur widely in [engineering](https://en.wikipedia.org/wiki/Engineering "Engineering"), [physics](https://en.wikipedia.org/wiki/Physics "Physics") and [mathematics](https://en.wikipedia.org/wiki/Mathematics "Mathematics").

### Galois theory

Given a polynomial, it may be that some of the roots are connected by various [algebraic equations](https://en.wikipedia.org/wiki/Algebraic_equation "Algebraic equation"). For example, it may be that for two of the roots, say *A* and *B*, that *A* <sup>2</sup> + 5 *B* <sup>3</sup> = 7. The central idea of Galois theory is to consider those [permutations](https://en.wikipedia.org/wiki/Permutation "Permutation") (or rearrangements) of the roots having the property that *any* algebraic equation satisfied by the roots is *still satisfied* after the roots have been permuted. An important proviso is that we restrict ourselves to algebraic equations whose coefficients are [rational numbers](https://en.wikipedia.org/wiki/Rational_number "Rational number"). Thus, Galois theory studies the symmetries inherent in algebraic equations.

### Automorphisms of algebraic objects

In [abstract algebra](https://en.wikipedia.org/wiki/Abstract_algebra "Abstract algebra"), an **automorphism** is an [isomorphism](https://en.wikipedia.org/wiki/Isomorphism "Isomorphism") from a [mathematical object](https://en.wikipedia.org/wiki/Mathematical_object "Mathematical object") to itself. It is, in some sense, a [symmetry](https://en.wikipedia.org/wiki/Symmetry "Symmetry") of the object, and a way of [mapping](https://en.wikipedia.org/wiki/Map_\(mathematics\) "Map (mathematics)") the object to itself while preserving all of its structure. The set of all automorphisms of an object forms a [group](https://en.wikipedia.org/wiki/Group_\(mathematics\) "Group (mathematics)"), called the **automorphism group**. It is, loosely speaking, the [symmetry group](https://en.wikipedia.org/wiki/Symmetry_group "Symmetry group") of the object.

#### Examples

- In [set theory](https://en.wikipedia.org/wiki/Set_theory "Set theory"), an arbitrary [permutation](https://en.wikipedia.org/wiki/Permutation "Permutation") of the elements of a set *X* is an automorphism. The automorphism group of *X* is also called the [symmetric group](https://en.wikipedia.org/wiki/Symmetric_group "Symmetric group") on *X*.
- In [elementary arithmetic](https://en.wikipedia.org/wiki/Elementary_arithmetic "Elementary arithmetic"), the set of [integers](https://en.wikipedia.org/wiki/Integer "Integer"), **Z**, considered as a group under addition, has a unique nontrivial automorphism: negation. Considered as a [ring](https://en.wikipedia.org/wiki/Ring_\(mathematics\) "Ring (mathematics)"), however, it has only the trivial automorphism. Generally speaking, negation is an automorphism of any [abelian group](https://en.wikipedia.org/wiki/Abelian_group "Abelian group"), but not of a ring or field.
- A group automorphism is a [group isomorphism](https://en.wikipedia.org/wiki/Group_isomorphism "Group isomorphism") from a group to itself. Informally, it is a permutation of the group elements such that the structure remains unchanged. For every group *G* there is a natural group homomorphism *G* → Aut(*G*) whose [image](https://en.wikipedia.org/wiki/Image_\(mathematics\) "Image (mathematics)") is the group Inn(*G*) of [inner automorphisms](https://en.wikipedia.org/wiki/Inner_automorphism "Inner automorphism") and whose [kernel](https://en.wikipedia.org/wiki/Kernel_\(algebra\) "Kernel (algebra)") is the [center](https://en.wikipedia.org/wiki/Center_\(group_theory\) "Center (group theory)") of *G*. Thus, if *G* has [trivial](https://en.wikipedia.org/wiki/Trivial_group "Trivial group") center it can be embedded into its own automorphism group.[^5]
- In [linear algebra](https://en.wikipedia.org/wiki/Linear_algebra "Linear algebra"), an endomorphism of a [vector space](https://en.wikipedia.org/wiki/Vector_space "Vector space") *V* is a [linear operator](https://en.wikipedia.org/wiki/Linear_transformation "Linear transformation") *V* → *V*. An automorphism is an invertible linear operator on *V*. When the vector space is finite-dimensional, the automorphism group of *V* is the same as the [general linear group](https://en.wikipedia.org/wiki/General_linear_group "General linear group"), GL(*V*).
- A field automorphism is a [bijective](https://en.wikipedia.org/wiki/Bijection "Bijection") [ring homomorphism](https://en.wikipedia.org/wiki/Ring_homomorphism "Ring homomorphism") from a [field](https://en.wikipedia.org/wiki/Field_\(mathematics\) "Field (mathematics)") to itself. In the cases of the [rational numbers](https://en.wikipedia.org/wiki/Rational_number "Rational number") (**Q**) and the [real numbers](https://en.wikipedia.org/wiki/Real_number "Real number") (**R**) there are no nontrivial field automorphisms. Some subfields of **R** have nontrivial field automorphisms, which however do not extend to all of **R** (because they cannot preserve the property of a number having a square root in **R**). In the case of the [complex numbers](https://en.wikipedia.org/wiki/Complex_number "Complex number"), **C**, there is a unique nontrivial automorphism that sends **R** into **R**: [complex conjugation](https://en.wikipedia.org/wiki/Complex_conjugate "Complex conjugate"), but there are infinitely ([uncountably](https://en.wikipedia.org/wiki/Uncountable "Uncountable")) many "wild" automorphisms (assuming the [axiom of choice](https://en.wikipedia.org/wiki/Axiom_of_choice "Axiom of choice")).[^6] Field automorphisms are important to the theory of [field extensions](https://en.wikipedia.org/wiki/Field_extension "Field extension"), in particular [Galois extensions](https://en.wikipedia.org/wiki/Galois_extension "Galois extension"). In the case of a Galois extension *L* / *K* the [subgroup](https://en.wikipedia.org/wiki/Subgroup "Subgroup") of all automorphisms of *L* fixing *K* pointwise is called the [Galois group](https://en.wikipedia.org/wiki/Galois_group "Galois group") of the extension.

## Symmetry in representation theory

### Symmetry in quantum mechanics: bosons and fermions

In quantum mechanics, bosons have representatives that are symmetric under permutation operators, and fermions have antisymmetric representatives.

This implies the Pauli exclusion principle for fermions. In fact, the Pauli exclusion principle with a single-valued many-particle wavefunction is equivalent to requiring the wavefunction to be antisymmetric. An antisymmetric two-particle state is represented as a [sum of states](https://en.wikipedia.org/wiki/Superposition_principle "Superposition principle") in which one particle is in state ${\displaystyle \scriptstyle |x\rangle }$ and the other in state ${\displaystyle \scriptstyle |y\rangle }$:

${\displaystyle |\psi \rangle =\sum _{x,y}A(x,y)|x,y\rangle }$

and antisymmetry under exchange means that *A* (*x*,*y*) = − *A* (*y*,*x*). This implies that *A* (*x*,*x*) = 0, which is Pauli exclusion. It is true in any basis, since unitary changes of basis keep antisymmetric matrices antisymmetric, although strictly speaking, the quantity *A* (*x*,*y*) is not a matrix but an antisymmetric rank-two [tensor](https://en.wikipedia.org/wiki/Tensor "Tensor").

Conversely, if the diagonal quantities *A* (*x*,*x*) are zero *in every basis*, then the wavefunction component:

${\displaystyle A(x,y)=\langle \psi |x,y\rangle =\langle \psi |(|x\rangle \otimes |y\rangle )}$

is necessarily antisymmetric. To prove it, consider the matrix element:

${\displaystyle \langle \psi |((|x\rangle +|y\rangle )\otimes (|x\rangle +|y\rangle ))\,}$

This is zero, because the two particles have zero probability to both be in the superposition state ${\displaystyle \scriptstyle |x\rangle +|y\rangle }$. But this is equal to

${\displaystyle \langle \psi |x,x\rangle +\langle \psi |x,y\rangle +\langle \psi |y,x\rangle +\langle \psi |y,y\rangle \,}$

The first and last terms on the right hand side are diagonal elements and are zero, and the whole sum is equal to zero. So the wavefunction matrix elements obey:

${\displaystyle \langle \psi |x,y\rangle +\langle \psi |y,x\rangle =0\,}$.

or

${\displaystyle A(x,y)=-A(y,x)\,}$

## Symmetry in set theory

### Symmetric relation

We call a relation symmetric if every time the relation stands from A to B, it stands too from B to A. Note that symmetry is not the exact opposite of [antisymmetry](https://en.wikipedia.org/wiki/Antisymmetric_relation "Antisymmetric relation").

## Symmetry in metric spaces

### Isometries of a space

An **isometry** is a [distance](https://en.wikipedia.org/wiki/Distance "Distance") -preserving map between [metric spaces](https://en.wikipedia.org/wiki/Metric_spaces "Metric spaces"). Given a metric space, or a set and scheme for assigning distances between elements of the set, an isometry is a transformation which maps elements to another metric space such that the distance between the elements in the new metric space is equal to the distance between the elements in the original metric space. In a two-dimensional or three-dimensional space, two geometric figures are [congruent](https://en.wikipedia.org/wiki/Congruence_\(geometry\) "Congruence (geometry)") if they are related by an isometry: related by either a [rigid motion](https://en.wikipedia.org/wiki/Rigid_body "Rigid body"), or a [composition](https://en.wikipedia.org/wiki/Function_composition "Function composition") of a rigid motion and a [reflection](https://en.wikipedia.org/wiki/Reflection_\(mathematics\) "Reflection (mathematics)"). Up to a relation by a rigid motion, they are equal if related by a [direct isometry](https://en.wikipedia.org/wiki/Euclidean_group#Direct_and_indirect_isometries "Euclidean group").

Isometries have been used to unify the working definition of symmetry in geometry and for functions, probability distributions, matrices, strings, graphs, etc.[^7]

## Symmetries of differential equations

A symmetry of a [differential equation](https://en.wikipedia.org/wiki/Differential_equation "Differential equation") is a transformation that leaves the differential equation invariant. Knowledge of such symmetries may help solve the differential equation.

A [Line symmetry](https://en.wikipedia.org/wiki/Line_symmetry "Line symmetry") of a [system of differential equations](https://en.wikipedia.org/wiki/System_of_differential_equations "System of differential equations") is a continuous symmetry of the system of differential equations. Knowledge of a Line symmetry can be used to simplify an ordinary differential equation through [reduction of order](https://en.wikipedia.org/wiki/Reduction_of_order "Reduction of order").[^8]

For [ordinary differential equations](https://en.wikipedia.org/wiki/Ordinary_differential_equations "Ordinary differential equations"), knowledge of an appropriate set of Lie symmetries allows one to explicitly calculate a set of first integrals, yielding a complete solution without integration.

Symmetries may be found by solving a related set of ordinary differential equations.[^8] Solving these equations is often much simpler than solving the original differential equations.

## Symmetry in probability

In the case of a finite number of possible outcomes, symmetry with respect to permutations (relabelings) implies a [discrete uniform distribution](https://en.wikipedia.org/wiki/Uniform_distribution_\(discrete\) "Uniform distribution (discrete)").

In the case of a real interval of possible outcomes, symmetry with respect to interchanging sub-intervals of equal length corresponds to a [continuous uniform distribution](https://en.wikipedia.org/wiki/Uniform_distribution_\(continuous\) "Uniform distribution (continuous)").

In other cases, such as "taking a random integer" or "taking a random real number", there are no probability distributions at all symmetric with respect to relabellings or to exchange of equally long subintervals. Other reasonable symmetries do not single out one particular distribution, or in other words, there is not a unique probability distribution providing maximum symmetry.

There is one type of [isometry in one dimension](https://en.wikipedia.org/wiki/Symmetry_groups_in_one_dimension "Symmetry groups in one dimension") that may leave the probability distribution unchanged, that is reflection in a point, for example zero.

A possible symmetry for randomness with positive outcomes is that the former applies for the logarithm, i.e., the outcome and its reciprocal have the same distribution. However this symmetry does not single out any particular distribution uniquely.
