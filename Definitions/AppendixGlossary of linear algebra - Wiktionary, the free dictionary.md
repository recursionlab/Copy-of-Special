## A

[affine transformation](https://en.wiktionary.org/wiki/affine_transformation "affine transformation")

A linear transformation between vector spaces followed by a translation.

## B

[basis](https://en.wiktionary.org/wiki/basis "basis")

In a vector space, a linearly independent set of vectors spanning the whole vector space.

## D

[determinant](https://en.wiktionary.org/wiki/determinant "determinant")

The unique scalar function over square matrices which is distributive over matrix multiplication, multilinear in the rows and columns, and takes the value of ${\displaystyle 1}$ for the unit matrix.

[diagonal matrix](https://en.wiktionary.org/wiki/diagonal_matrix "diagonal matrix")

A matrix in which only the entries on the main diagonal are non-zero.

[dimension](https://en.wiktionary.org/wiki/dimension "dimension")

The number of elements of any basis of a vector space.

## I

[identity matrix](https://en.wiktionary.org/wiki/identity_matrix "identity matrix")

A diagonal matrix all of the diagonal elements of which are equal to ${\displaystyle 1}$.

[inverse matrix](https://en.wiktionary.org/wiki/inverse_matrix "inverse matrix")

Of a matrix ${\displaystyle A}$, another matrix ${\displaystyle B}$ such that ${\displaystyle A}$ multiplied by ${\displaystyle B}$ and ${\displaystyle B}$ multiplied by ${\displaystyle A}$ both equal the identity matrix.

## L

[linear algebra](https://en.wiktionary.org/wiki/linear_algebra "linear algebra")

The branch of mathematics that deals with vectors, vector spaces, linear transformations and systems of linear equations.

[linear combination](https://en.wiktionary.org/wiki/linear_combination "linear combination")

A sum, each of whose summands is an appropriate vector times an appropriate scalar (or ring element).

[linear equation](https://en.wiktionary.org/wiki/linear_equation "linear equation")

A polynomial equation of the first degree (such as ${\displaystyle x=2y-7}$).

[linear transformation](https://en.wiktionary.org/wiki/linear_transformation "linear transformation")

A map between vector spaces which respects addition and multiplication.

[linearly independent](https://en.wiktionary.org/wiki/linearly_independent "linearly independent")

(Of a set of vectors or ring elements) whose nontrivial linear combinations are nonzero.

## M

[matrix](https://en.wiktionary.org/wiki/matrix "matrix")

A rectangular arrangement of numbers or terms having various uses such as transforming coordinates in geometry, solving systems of linear equations in linear algebra and representing graphs in graph theory.

## S

[spectrum](https://en.wiktionary.org/wiki/spectrum "spectrum")

Of a bounded linear operator ${\displaystyle A}$, the scalar values ${\displaystyle \lambda }$ such that the operator ${\displaystyle A-\lambda I}$, where ${\displaystyle I}$ denotes the identity operator, does not have a bounded inverse.

[square matrix](https://en.wiktionary.org/wiki/square_matrix "square matrix")

A matrix having the same number of rows as columns.

## V

[vector](https://en.wiktionary.org/wiki/vector "vector")

A directed quantity, one with both magnitude and direction; an element of a vector space.

[vector space](https://en.wiktionary.org/wiki/vector_space "vector space")

A set ${\displaystyle V}$, whose elements are called "vectors", together with a binary operation ${\displaystyle +}$ forming a [module](https://en.wiktionary.org/wiki/module "module") ${\displaystyle (V,+)}$, and a set ${\displaystyle F^{*}}$ of [bilinear](https://en.wiktionary.org/wiki/bilinear "bilinear") unary functions ${\displaystyle f^{*}:V\rightarrow V}$, each of which corresponds to a " [scalar](https://en.wiktionary.org/wiki/scalar "scalar") " element ${\displaystyle f}$ of a [field](https://en.wiktionary.org/wiki/field "field") ${\displaystyle F}$, such that the composition of elements of ${\displaystyle F^{*}}$ corresponds isomorphically to multiplication of elements of ${\displaystyle F}$, and such that for any vector ${\displaystyle \mathbf {v} ,1^{*}(\mathbf {v} )=\mathbf {v} }$.
