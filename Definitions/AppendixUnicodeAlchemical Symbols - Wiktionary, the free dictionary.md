## Alchemical Symbols

This page lists the characters in the [Alchemical Symbols](http://unicode.org/charts/PDF/U1F700.pdf) block of the [Unicode](https://en.wikipedia.org/wiki/Unicode "w:Unicode") standard (version 17.0), which covers 128 code points from U+1F700 to U+1F77F, all of which have been assigned.

All characters in this block have the `General_Category` value `So` (`Other_Symbol`) and the `Script` value `Zyyy` (`Common`).
