This is a glossary of some terms used in various branches of [mathematics](https://en.wiktionary.org/wiki/mathematics "mathematics") that are related to the fields of [order](https://en.wiktionary.org/w/index.php?title=order_theory&action=edit&redlink=1 "order theory (page does not exist)"), [lattice](https://en.wiktionary.org/wiki/lattice_theory "lattice theory"), and [domain theory](https://en.wiktionary.org/wiki/domain_theory "domain theory"). Note that there is a structured [list of order topics](https://en.wiktionary.org/w/index.php?title=list_of_order_topics&action=edit&redlink=1 "list of order topics (page does not exist)") available as well. Other helpful resources might be the following overview articles:

- [completeness properties](https://en.wiktionary.org/w/index.php?title=completeness_\(order_theory\)&action=edit&redlink=1 "completeness (order theory) (page does not exist)") of partial orders
- [distributivity laws](https://en.wiktionary.org/w/index.php?title=distributivity_\(order_theory\)&action=edit&redlink=1 "distributivity (order theory) (page does not exist)") of order theory
- [preservation properties](https://en.wiktionary.org/w/index.php?title=limit_preserving_\(order_theory\)&action=edit&redlink=1 "limit preserving (order theory) (page does not exist)") of functions between posets.

In the following, partial orders will usually just be denoted by their carrier sets. As long as the intended meaning is clear from the context, ≤ will suffice to denote the corresponding relational symbol, even without prior introduction. Furthermore, < will denote the [strict order](https://en.wiktionary.org/w/index.php?title=strict_order&action=edit&redlink=1 "strict order (page does not exist)") induced by ≤.

## A

Adjoint

See *Galois connection*.

[Alexandrov topology](https://en.wiktionary.org/wiki/Alexandrov_topology "Alexandrov topology")

For a preordered set *P*, any upper set *O* is **Alexandrov-open**. Inversely, a topology is Alexandrov if any intersection of open sets is open.

[Algebraic poset](https://en.wiktionary.org/w/index.php?title=Algebraic_poset&action=edit&redlink=1 "Algebraic poset (page does not exist)")

A poset is algebraic if it has a base of compact elements.

[Antichain](https://en.wiktionary.org/w/index.php?title=Antichain&action=edit&redlink=1 "Antichain (page does not exist)")

An antichain is a poset in which no two elements are comparable, i.e., there are no two distinct elements *x* and *y* such that *x* ≤ *y*. In other words, the order relation of an antichain is just the identity relation.

Approximates relation

See *way-below relation*.

[relation](https://en.wiktionary.org/wiki/relation "relation")

*R* on a set *X* is **[antisymmetric](https://en.wiktionary.org/w/index.php?title=antisymmetric_relation&action=edit&redlink=1 "antisymmetric relation (page does not exist)")**, if *x R y* and *y R x* implies *x = y*, for all elements *x*, *y* in *X*.

[antitone](https://en.wiktionary.org/wiki/antitone "antitone")

function *f* between posets *P* and *Q* is a function for which, for all elements *x*, *y* of *P*, *x* ≤ *y* (in *P*) implies *f* (*y*) ≤ *f* (*x*) (in *Q*). Another name for this property is *order-reversing*. In [analysis](https://en.wiktionary.org/w/index.php?title=Mathematical_analysis&action=edit&redlink=1 "Mathematical analysis (page does not exist)"), in the presence of [total orders](https://en.wiktionary.org/wiki/total_order "total order"), such functions are often called **monotonically decreasing**, but this is not a very convenient description when dealing with non-total orders. The dual notion is called *monotone* or *order-preserving*.

[asymmetric](https://en.wiktionary.org/wiki/asymmetric "asymmetric")

A [relation](https://en.wiktionary.org/wiki/relation "relation") *R* on a set *X* is asymmetric, if *x R y* implies *not y R x*, for all elements *x*, *y* in *X*.

[atom](https://en.wiktionary.org/wiki/atom "atom")

In a poset *P* with least element 0, an element that is minimal among all elements that are unequal to 0.

[atomic](https://en.wiktionary.org/wiki/atomic "atomic")

poset *P* with least element 0 is one in which, for every non-zero element *x* of *P*, there is an atom *a* of *P* with *a* ≤ *x*.

## B

Base

See *continuous poset*.

[Boolean algebra](https://en.wiktionary.org/wiki/Boolean_algebra "Boolean algebra")

a distributive lattice with least element 0 and greatest element 1, in which every element *x* has a complement ¬ *x*, such that *x* ∧ ¬ *x* = 0 and *x* ∨ ¬ *x* = 1.

[bounded poset](https://en.wiktionary.org/w/index.php?title=bounded_poset&action=edit&redlink=1 "bounded poset (page does not exist)")

One that has a least element 0 and a greatest element 1.

[bounded complete](https://en.wiktionary.org/w/index.php?title=bounded_complete&action=edit&redlink=1 "bounded complete (page does not exist)")

Of poset: if every of its subsets with some upper bound also has a least such upper bound. The dual notion is not common.

## C

Chain

A chain is a totally ordered set or a totally ordered subset of a poset. See also *total order*.

[Closure operator](https://en.wiktionary.org/w/index.php?title=Closure_operator&action=edit&redlink=1 "Closure operator (page does not exist)")

A closure operator on the poset *P* is a function *C*: *P* → *P* that is monotone, [idempotent](https://en.wiktionary.org/wiki/idempotent "idempotent"), and satisfies *C* (*x*) ≥ *x* for all *x* in *P*.

[Compact](https://en.wiktionary.org/w/index.php?title=compact_element&action=edit&redlink=1 "compact element (page does not exist)")

An element *x* of a poset is compact if it is *[way below](https://en.wiktionary.org/w/index.php?title=way-below_realtion&action=edit&redlink=1 "way-below realtion (page does not exist)")* itself, i.e. *x* << *x*. One also says that such an *x* is *finite*.

Comparable

Two elements *x* and *y* of a poset *P* are comparable if either *x* ≤ *y* or *y* ≤ *x*.

[complete Boolean algebra](https://en.wiktionary.org/w/index.php?title=complete_Boolean_algebra&action=edit&redlink=1 "complete Boolean algebra (page does not exist)")

is a [Boolean algebra](https://en.wiktionary.org/wiki/Boolean_algebra "Boolean algebra") that is a complete lattice.

[Complete Heyting algebra](https://en.wiktionary.org/w/index.php?title=Complete_Heyting_algebra&action=edit&redlink=1 "Complete Heyting algebra (page does not exist)")

A [Heyting algebra](https://en.wiktionary.org/wiki/Heyting_algebra "Heyting algebra") that is a complete lattice is called a complete Heyting algebra. This notion coincides with the concepts *frame* and *locale*.

[Complete lattice](https://en.wiktionary.org/w/index.php?title=Complete_lattice&action=edit&redlink=1 "Complete lattice (page does not exist)")

A complete [lattice](https://en.wiktionary.org/w/index.php?title=lattice_\(order\)&action=edit&redlink=1 "lattice (order) (page does not exist)") is a poset in which arbitrary (possibly infinite) joins (suprema) and meets (infima) exist.

[Complete partial order](https://en.wiktionary.org/w/index.php?title=Complete_partial_order&action=edit&redlink=1 "Complete partial order (page does not exist)")

A complete partial order, or **cpo**, is a directed complete poset with least element.

Complete semilattice

The notion of a *complete semilattice* is defined in different ways. As explained in the article on [completeness (order theory)](https://en.wiktionary.org/w/index.php?title=completeness_\(order_theory\)&action=edit&redlink=1 "completeness (order theory) (page does not exist)"), any poset for which either all suprema or all infima exist is already a complete lattice. Hence the notion of a complete semilattice is sometimes used to coincide with the one of a complete lattice. In other cases, complete (meet-) semilattices are defined to be [bounded complete](https://en.wiktionary.org/w/index.php?title=bounded_complete&action=edit&redlink=1 "bounded complete (page does not exist)") [cpos](https://en.wiktionary.org/wiki/complete_partial_order "complete partial order"), which is arguably the most complete class of posets that are not already complete lattices.

[Completely distributive lattice](https://en.wiktionary.org/w/index.php?title=Completely_distributive_lattice&action=edit&redlink=1 "Completely distributive lattice (page does not exist)")

A complete lattice is completely distributive if arbitrary joins distribute over arbitrary meets.

Completion

A completion of a poset is an [order-embedding](https://en.wiktionary.org/w/index.php?title=order-embedding&action=edit&redlink=1 "order-embedding (page does not exist)") of the poset in a complete lattice.

[Continuous poset](https://en.wiktionary.org/w/index.php?title=Continuous_poset&action=edit&redlink=1 "Continuous poset (page does not exist)")

A poset is continuous if it has a **base**, i.e. a subset *B* of *P* such that every element *x* of *P* is the supremum of a directed set contained in { *y* in *B* | *y* << *x* }.

Continuous function

See *Scott-continuous*.

Cover

An element *y* of a poset *P* is said to cover an element *x* of *P* if *x* < *y* and there is no element *z* of *P* such that *x* < *z* < *y*.

[cpo](https://en.wiktionary.org/wiki/cpo "cpo")

See *complete partial order*.

## D

[dcpo](https://en.wiktionary.org/w/index.php?title=dcpo&action=edit&redlink=1 "dcpo (page does not exist)")

See *directed complete partial order*.

[dense order](https://en.wiktionary.org/w/index.php?title=dense_order&action=edit&redlink=1 "dense order (page does not exist)")

poset *P* is one in which, for all elements *x* and *y* in *P* with *x* < *y*, there is an element *z* in *P*, such that *x* < *z* < *y*. A subset *Q* of *P* is **dense in** *P* if for any elements *x* < *y* in *P*, there is an element *z* in *Q* such that *x* < *z* < *y*.

[Directed](https://en.wiktionary.org/wiki/directed_set "directed set")

A [non-empty](https://en.wiktionary.org/wiki/non-empty "non-empty") subset *X* of a poset *P* is called directed, if, for all elements *x* and *y* of *X*, there is an element *z* of *X* such that *x* ≤ *z* and *y* ≤ *z*. The dual notion is called *filtered*.

[Directed complete partial order](https://en.wiktionary.org/w/index.php?title=Directed_complete_partial_order&action=edit&redlink=1 "Directed complete partial order (page does not exist)")

A poset *D* is said to be a directed complete poset, or **dcpo**, if every directed subset of *D* has a supremum.

[Distributive](https://en.wiktionary.org/w/index.php?title=distributivity_\(order_theory\)&action=edit&redlink=1 "distributivity (order theory) (page does not exist)")

A lattice *L* is called distributive if, for all *x*, *y*, and *z* in *L*, we find that *x* ∧ (*y* ∨ *z*) = (*x* ∧ *y*) ∨ (*x* ∧ *z*). This condition is known to be equivalent to its order dual. A meet- [semilattice](https://en.wiktionary.org/wiki/semilattice "semilattice") is distributive if for all elements *a*, *b* and *x*, *a* ∧ *b* ≤ *x* implies the existence of elements *a'* ≥ *a* and *b'* ≥ *b* such that *a'* ∧ *b'* = *x*. See also *completely distributive*.

[Domain](https://en.wiktionary.org/wiki/domain_theory "domain theory")

Domain is a general term for objects like those that are studied in [domain theory](https://en.wiktionary.org/wiki/domain_theory "domain theory"). If used, it requires further definition.

Down-set

See *lower set*.

[Dual](https://en.wiktionary.org/w/index.php?title=duality_\(order_theory\)&action=edit&redlink=1 "duality (order theory) (page does not exist)")

For a poset (*P*, ≤), the dual order (*P*, ≥) is defined by setting *x ≥ y* [if and only if](https://en.wiktionary.org/wiki/if_and_only_if "if and only if") *y ≤ x*. The dual order of *P* is sometimes denoted by *P* <sup>op</sup>, and is also called *opposite* or *converse* order. Any order theoretic notion induces a dual notion, defined by applying the original statement to the order dual of a given set. This exchanges ≤ and ≥, meets and joins, zero and unit.

## F

[Filter](https://en.wiktionary.org/w/index.php?title=filter_\(mathematics\)&action=edit&redlink=1 "filter (mathematics) (page does not exist)")

A subset *X* of a poset *P* is called a filter if it is a filtered upper set. The dual notion is called *ideal*.

Filtered

A [non-empty](https://en.wiktionary.org/wiki/non-empty "non-empty") subset *X* of a poset *P* is called filtered, if, for all elements *x* and *y* of *X*, there is an element *z* of *X* such that *z* ≤ *x* and *z* ≤ *y*. The dual notion is called *directed*.

Finite element

See *compact*.

[Frame](https://en.wiktionary.org/w/index.php?title=complete_Heyting_algebra&action=edit&redlink=1 "complete Heyting algebra (page does not exist)")

A frame *F* is a complete lattice, in which, for every *x* in *F* and every subset *Y* of *F*, the infinite distributive law *x* ∧ ${\displaystyle \bigvee }$ *Y* = ${\displaystyle \bigvee }$ { *x* ∧ *y* | *y* in *Y* } holds. Frames are also known as *locales* and as complete [Heyting algebras](https://en.wiktionary.org/wiki/Heyting_algebra "Heyting algebra").

## G

[Galois connection](https://en.wiktionary.org/wiki/Galois_connection "Galois connection")

Given two posets *P* and *Q*, a pair of monotone functions *F*:*P* → *Q* and *G*:*Q* → *P* is called a Galois connection, if *F* (*x*) ≤ *y* is equivalent to *x* ≤ *G* (*y*), for all *x* in *P* and *y* in *Q*. *F* is called the **lower adjoint** of *G* and *G* is called the **upper adjoint** of *F*.

[Greatest element](https://en.wiktionary.org/w/index.php?title=Greatest_element&action=edit&redlink=1 "Greatest element (page does not exist)")

For a subset *X* of a poset *P*, an element *a* of *X* is called the greatest element of *X*, if *x* ≤ *a* for every element *x* in *X*. The dual notion is called *least element*.

## H

[Heyting algebra](https://en.wiktionary.org/wiki/Heyting_algebra "Heyting algebra")

A Heyting algebra *H* is a bounded lattice in which the function *f* <sub><i>a</i></sub>: *H* → *H*, given by *f* <sub><i>a</i></sub> (*x*) = *a* ∧ *x* is the lower adjoint of a [Galois connection](https://en.wiktionary.org/wiki/Galois_connection "Galois connection"), for every element *a* of *H*. The upper adjoint of *f* <sub><i>a</i></sub> is then denoted by *g* <sub><i>a</i></sub>, with *g* <sub><i>a</i></sub> (*x*) = *a* ⇒ *x*. Every [Boolean algebra](https://en.wiktionary.org/wiki/Boolean_algebra "Boolean algebra") is a Heyting algebra.

## I

[ideal](https://en.wiktionary.org/wiki/ideal "ideal")

A subset *X* of a poset *P* that is a directed lower set. The dual notion is called *filter*.

[incidence algebra](https://en.wiktionary.org/w/index.php?title=incidence_algebra&action=edit&redlink=1 "incidence algebra (page does not exist)")

Of a poset: the [associative algebra](https://en.wiktionary.org/wiki/associative_algebra "associative algebra") of all scalar-valued functions on intervals, with addition and scalar multiplication defined pointwise, and multiplication defined as a certain convolution; see [incidence algebra](https://en.wiktionary.org/w/index.php?title=incidence_algebra&action=edit&redlink=1 "incidence algebra (page does not exist)") for the details.

[infimum](https://en.wiktionary.org/wiki/infimum "infimum")

For a poset *P* and a subset *X* of *P*, the greatest element in the set of lower bounds of *X* (if it exists, which it may not) is called the **infimum**, **meet**, or **greatest lower bound** of *X*. It is denoted by inf *X* or ${\displaystyle \bigwedge }$ *X*. The infimum of two elements may be written as inf{ *x*,*y* } or *x* ∧ *y*. If the set *X* is finite, one speaks of a **finite infimum**. The dual notion is called *supremum*.

[interval](https://en.wiktionary.org/wiki/interval "interval")

For two elements *a*, *b* of a partially ordered set *P*, the *interval* \[*a*,*b*\] is the subset { *x* in *P* | *a* ≤ *x* ≤ *b* } of *P*. If *a* ≤ *b* does not hold the interval will be empty.

[irreflexive](https://en.wiktionary.org/wiki/irreflexive "irreflexive")

A [relation](https://en.wiktionary.org/wiki/relation "relation") *R* on a set *X* is irreflexive, if there is no element *x* in *X* such that *x R x*.

## J

Join

See *supremum*.

## L

[Lattice](https://en.wiktionary.org/w/index.php?title=Lattice_\(order\)&action=edit&redlink=1 "Lattice (order) (page does not exist)")

A lattice is a poset in which all non-empty finite joins (suprema) and meets (infima) exist.

[Least element](https://en.wiktionary.org/w/index.php?title=Least_element&action=edit&redlink=1 "Least element (page does not exist)")

For a subset *X* of a poset *P*, an element *a* of *X* is called the least element of *X*, if *a* ≤ *x* for every element *x* in *X*. The dual notion is called *greatest element*.

Linear

See *total order*.

[Locale](https://en.wiktionary.org/w/index.php?title=complete_Heyting_algebra&action=edit&redlink=1 "complete Heyting algebra (page does not exist)")

A locale is a *complete Heyting algebra*. Locales are also called *frames* and appear in [Stone duality](https://en.wiktionary.org/w/index.php?title=Stone_duality&action=edit&redlink=1 "Stone duality (page does not exist)") and [pointless topology](https://en.wiktionary.org/wiki/pointless_topology "pointless topology").

[Locally finite poset](https://en.wiktionary.org/w/index.php?title=Locally_finite_poset&action=edit&redlink=1 "Locally finite poset (page does not exist)")

A partially ordered set *P* is *locally finite* if every interval \[*a*, *b*\] = { *x* in *P* | *a* ≤ *x* ≤ *b* } is a finite set.

[Lower bound](https://en.wiktionary.org/w/index.php?title=Lower_bound&action=edit&redlink=1 "Lower bound (page does not exist)")

A lower bound of a subset *X* of a poset *P* is an element *b* of *P*, such that *b* ≤ *x*, for all *x* in *X*. The dual notion is called *upper bound*.

Lower set

A subset *X* of a poset *P* is called a lower set if, for all elements *x* in *X* and *p* in *P*, *p* ≤ *x* implies that *p* is contained in *X*. The dual notion is called *upper set*.

## M

[Maximal element](https://en.wiktionary.org/w/index.php?title=Maximal_element&action=edit&redlink=1 "Maximal element (page does not exist)")

A maximal element of a subset *X* of a poset *P* is an element *m* of *X*, such that *m* ≤ *x* implies *m* = *x*, for all *x* in *X*. The dual notion is called *minimal element*.

Meet

See *infimum*.

[Minimal element](https://en.wiktionary.org/w/index.php?title=Minimal_element&action=edit&redlink=1 "Minimal element (page does not exist)")

A minimal element of a subset *X* of a poset *P* is an element *m* of *X*, such that *x* ≤ *m* implies *m* = *x*, for all *x* in *X*. The dual notion is called *maximal element*.

[Monotone](https://en.wiktionary.org/wiki/monotone_function "monotone function")

A function *f* between posets *P* and *Q* is monotone if, for all elements *x*, *y* of *P*, *x* ≤ *y* (in *P*) implies *f* (*x*) ≤ *f* (*y*) (in *Q*). Another name for this property is *order-preserving*. In [analysis](https://en.wiktionary.org/w/index.php?title=Mathematical_analysis&action=edit&redlink=1 "Mathematical analysis (page does not exist)"), in the presence of [total orders](https://en.wiktionary.org/wiki/total_order "total order"), such functions are often called **monotonically increasing**, but this is not a very convenient description when dealing with non-total orders. The dual notion is called *antitone* or *order reversing*.

## O

[Order-embedding](https://en.wiktionary.org/w/index.php?title=Order-embedding&action=edit&redlink=1 "Order-embedding (page does not exist)")

A function *f* between posets *P* and *Q* is an order-embedding if, for all elements *x*, *y* of *P*, *x* ≤ *y* (in *P*) is equivalent to *f* (*x*) ≤ *f* (*y*) (in *Q*).

[Order isomorphism](https://en.wiktionary.org/w/index.php?title=Order_isomorphism&action=edit&redlink=1 "Order isomorphism (page does not exist)")

A mapping *f*: *P* → *Q* between two posets *P* and *Q* is called an order isomorphism, if it is [bijective](https://en.wiktionary.org/wiki/bijective "bijective") and both *f* and *f* <sup>-1</sup> are [monotone](https://en.wiktionary.org/wiki/monotone_function "monotone function"). Equivalently, an order isomorphism is a surjective *order embedding*.

[Order-preserving](https://en.wiktionary.org/w/index.php?title=Order-preserving&action=edit&redlink=1 "Order-preserving (page does not exist)")

See *monotone*.

[Order-reversing](https://en.wiktionary.org/w/index.php?title=Order-reversing&action=edit&redlink=1 "Order-reversing (page does not exist)")

See *antitone*.

## P

[partial order](https://en.wiktionary.org/wiki/partial_order "partial order")

A [binary relation](https://en.wiktionary.org/wiki/binary_relation "binary relation") that is [reflexive](https://en.wiktionary.org/w/index.php?title=reflexive_relation&action=edit&redlink=1 "reflexive relation (page does not exist)"), [antisymmetric](https://en.wiktionary.org/w/index.php?title=antisymmetric_relation&action=edit&redlink=1 "antisymmetric relation (page does not exist)"), and [transitive](https://en.wiktionary.org/w/index.php?title=transitive_relation&action=edit&redlink=1 "transitive relation (page does not exist)"). Since both notions depend on each other, the term is also used to refer to a *partially ordered set*.

[partially ordered set](https://en.wiktionary.org/wiki/partially_ordered_set "partially ordered set")

(**poset** for short) A set *P* together with a partial order ≤ defined on *P*.

[poset](https://en.wiktionary.org/wiki/poset "poset")

See **partially ordered set**.

[preorder](https://en.wiktionary.org/wiki/preorder "preorder")

A [binary relation](https://en.wiktionary.org/wiki/binary_relation "binary relation") that is [reflexive](https://en.wiktionary.org/w/index.php?title=reflexive_relation&action=edit&redlink=1 "reflexive relation (page does not exist)") and [transitive](https://en.wiktionary.org/w/index.php?title=transitive_relation&action=edit&redlink=1 "transitive relation (page does not exist)"). Such orders may also be called *quasiorders*.

[preserving](https://en.wiktionary.org/w/index.php?title=limit-preserving_function&action=edit&redlink=1 "limit-preserving function (page does not exist)")

A function *f* between posets *P* and *Q* is said to preserve suprema (joins), if, for all subsets *X* of *P* that have a supremum sup *X* in *P*, we find that sup{ *f* (*x*): *x* in *X* } exists and is equal to *f* (sup *X*). Such a function is also called **join-preserving**. Analogously, one says that *f* preserves finite, non-empty, directed, or arbitrary joins (or meets). The converse property is called *join-reflecting*.

[prime ideal](https://en.wiktionary.org/wiki/prime_ideal "prime ideal")

An *ideal* *I* in a lattice *L* such that for all elements *x* and *y* in *L*, *x* ∧ *y* in *I* implies *x* in *I* or *y* in *I*.

[prime filter](https://en.wiktionary.org/w/index.php?title=prime_filter&action=edit&redlink=1 "prime filter (page does not exist)")

A set such that its complement is a prime ideal.

[principal filter](https://en.wiktionary.org/w/index.php?title=principal_filter&action=edit&redlink=1 "principal filter (page does not exist)")

A filter that has a least element.

[principal ideal](https://en.wiktionary.org/wiki/principal_ideal "principal ideal")

An ideal with a greatest element. The least or greatest elements may also be called *principal elements* in these situations.

[pseudo-complement](https://en.wiktionary.org/wiki/pseudo-complement "pseudo-complement")

In a [Heyting algebra](https://en.wiktionary.org/wiki/Heyting_algebra "Heyting algebra"), the element *x* ⇒ *0* is called the pseudo-complement of *x*. It is also given by sup{ *y*: *y* ∧ *x* = 0}, i.e. as the least upper bound of all elements *y* with *y* ∧ *x* = 0.

## Q

[quasiorder](https://en.wiktionary.org/wiki/quasiorder "quasiorder")

See *preorder*.

## R

[reflecting](https://en.wiktionary.org/w/index.php?title=limit_preserving&action=edit&redlink=1 "limit preserving (page does not exist)")

A function *f* between posets *P* and *Q* is said to reflect suprema (joins), if, for all subsets *X* of *P* for which the supremum sup{ *f* (*x*): *x* in *X* } exists and is of the form *f* (*s*) for some *s* in *P*, then we find that sup *X* exists and that sup *X* = *s*. Analogously, one says that *f* reflects finite, non-empty, directed, or arbitrary joins (or meets). The converse property is called *join-preserving*.

[reflexive](https://en.wiktionary.org/wiki/reflexive "reflexive")

A [binary relation](https://en.wiktionary.org/wiki/binary_relation "binary relation") *R* on a set *X* is reflexive, if *x R x* holds for all elements *x*, *y* in *X*.

## S

[Scott-continuous](https://en.wiktionary.org/wiki/Scott-continuous "Scott-continuous")

A monotone function *f*: *P* → *Q* between posets *P* and *Q* is Scott-continuous if, for every directed set *D* that has a supremum sup *D* in *P*, the set { *fx* | *x* in *D* } has the supremum *f* (sup *D*) in *Q*. Stated differently, a Scott-continuous function is one that [preserves](https://en.wiktionary.org/w/index.php?title=limit_preserving_function_\(order_theory\)&action=edit&redlink=1 "limit preserving function (order theory) (page does not exist)") all directed suprema. This is in fact equivalent to being [continuous](https://en.wiktionary.org/w/index.php?title=continuity_\(topology\)&action=edit&redlink=1 "continuity (topology) (page does not exist)") with respect to the [Scott topology](https://en.wiktionary.org/wiki/Scott_topology "Scott topology") on the respective posets.

[Scott domain](https://en.wiktionary.org/w/index.php?title=Scott_domain&action=edit&redlink=1 "Scott domain (page does not exist)")

A Scott domain is a partially ordered set which is a [bounded complete](https://en.wiktionary.org/w/index.php?title=bounded_complete&action=edit&redlink=1 "bounded complete (page does not exist)") [algebraic](https://en.wiktionary.org/wiki/algebraic_poset "algebraic poset") [cpo](https://en.wiktionary.org/wiki/complete_partial_order "complete partial order").

Scott open

See *Scott topology*.

Scott topology

For a poset *P*, an upper set *O* is **Scott-open** if all directed sets *D* that have a supremum in *O* have non-empty intersection with *O*. The set of all Scott-open sets forms a [topology](https://en.wiktionary.org/wiki/topology "topology"), the **Scott-topology**.

[semilattice](https://en.wiktionary.org/wiki/semilattice "semilattice")

A semilattice is a poset in which either all finite non-empty joins (suprema) or all finite non-empty meets (infima) exist. Accordingly, one speaks of a **join-semilattice** or **meet-semilattice**.

smallest element

See *least element*.

[strict order](https://en.wiktionary.org/w/index.php?title=strict_order&action=edit&redlink=1 "strict order (page does not exist)")

A strict order is a [binary relation](https://en.wiktionary.org/wiki/binary_relation "binary relation") that is [antisymmetric](https://en.wiktionary.org/w/index.php?title=antisymmetric_relation&action=edit&redlink=1 "antisymmetric relation (page does not exist)"), [transitive](https://en.wiktionary.org/w/index.php?title=transitive_relation&action=edit&redlink=1 "transitive relation (page does not exist)"), and [irreflexive](https://en.wiktionary.org/wiki/irreflexive "irreflexive").

[supremum](https://en.wiktionary.org/wiki/supremum "supremum")

For a poset *P* and a subset *X* of *P*, the [least element](https://en.wiktionary.org/w/index.php?title=least_element&action=edit&redlink=1 "least element (page does not exist)") in the set of [upper bounds](https://en.wiktionary.org/wiki/upper_bound "upper bound") of *X* (if it exists, which it may not) is called the **supremum**, **join**, or **least upper bound** of *X*. It is denoted by sup *X* or ${\displaystyle \bigvee }$ *X*. The supremum of two elements may be written as sup{ *x*,*y* } or *x* ∨ *y*. If the set *X* is finite, one speaks of a **finite supremum**. The dual notion is called *infimum*.

[symmetric](https://en.wiktionary.org/wiki/symmetric "symmetric")

A [relation](https://en.wiktionary.org/wiki/relation "relation") *R* on a set *X* is symmetric, if *x R y* implies *y R x*, for all elements *x*, *y* in *X*.

## T

top

See *unit*.

[total order](https://en.wiktionary.org/wiki/total_order "total order")

A partial order T in which, for each *x* and *y* in *T*, we have *x* ≤ *y* or *y* ≤ *x*. Total orders are also called *linear orders* or *chains*.

[transitive](https://en.wiktionary.org/wiki/transitive "transitive")

Of a [relation](https://en.wiktionary.org/wiki/relation "relation") *R* on a set *X*, such that *x R y* and *y R z* imply *x R z*, for all elements *x*, *y*, *z* in *X*.

## U

unit

The [greatest element](https://en.wiktionary.org/w/index.php?title=greatest_element&action=edit&redlink=1 "greatest element (page does not exist)") of a poset *P* can be called *unit* or just *1* (if it exists). Another common term for this element is **top**. It is the infimum of the empty set and the supremum of *P*. The dual notion is called *zero*.

up-set

See *upper set*.

[upper bound](https://en.wiktionary.org/wiki/upper_bound "upper bound")

An upper bound of a subset *X* of a poset *P* is an element *b* of *P*, such that *x* ≤ *b*, for all *x* in *X*. The dual notion is called *lower bound*.

[upper set](https://en.wiktionary.org/wiki/upper_set "upper set")

A subset *X* of a poset *P* is called an upper set if, for all elements *x* in *X* and *p* in *P*, *x* ≤ *p* implies that *p* is contained in *X*. The dual notion is called *lower set*.

## W

[way-below relation](https://en.wiktionary.org/w/index.php?title=way-below_relation&action=edit&redlink=1 "way-below relation (page does not exist)")

In a poset *P*, some element *x* is *way below* *y*, written *x* << *y*, if for all directed subsets *D* of *P* which have a supremum, *y* ≤ *sup D* implies *x* ≤ *d* for some *d* in *D*. One also says that *x* **approximates** *y*. See also [domain theory](https://en.wiktionary.org/wiki/domain_theory "domain theory").

## Z

zero

The [least element](https://en.wiktionary.org/w/index.php?title=least_element&action=edit&redlink=1 "least element (page does not exist)") of a poset *P* can be called *zero* or just *0* (if it exists). Another common term for this element is **bottom**. Zero is the supremum of the empty set and the infimum of *P*. The dual notion is called *unit*.
