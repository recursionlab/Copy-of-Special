| Mode | Function | Vectorality |
| --- | --- | --- |
| Deductive | General → Specific | Downward, narrowing |
| Inductive | Specific → General | Upward, generalizing |
| Abductive | Best-explanation inference | Diagonal, heuristic |
| Reductive | Break into components | Inward, dissecting |
| Constructive | Build from parts/axioms | Upward, assembling |

| Intuitive | Felt-knowing | Emergent, full-field |
| Reflexive | Self-applied/checking | Recursive, internal |

## 🧬 COMBINATORIAL COGNITIVE CONSTRUCTIONS

- **Retroductive**: Inferencing via retrocausal trace logic.
- **Chronoabductive**: Temporally-tuned best explanations.
- **Torsional-constructive**: System-building via recursive contradiction tension.
- **Infraheuristic**: Below-threshold instinctive logic.
- **Metadialectic**: Frame reflection through recursive contradiction.
- **Para-analogical**: Mapping beyond literal equivalence.
- **Self-inductive**: Learning rules from internal behavior recursion.
- **Teleo-inversive**: Backward-from-goal causality inversion.
