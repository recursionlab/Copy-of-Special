



Here is a list of common mathematical symbols that are generally considered non-ambiguous, meaning they have a single, well-defined meaning in standard mathematical notation. Note that context can sometimes alter interpretation, but these symbols are widely accepted with the given definitions.

### Arithmetic and Basic Operations
- **+** : Addition (e.g., 2 + 3 = 5)
- **−** : Subtraction (e.g., 5 − 2 = 3)
- **×** : Multiplication (e.g., 3 × 4 = 12; also used as cross product in vectors)
- **÷** : Division (e.g., 10 ÷ 2 = 5)
- **=** : Equality (e.g., x = y)
- **≠** : Inequality (not equal)
- **<** : Less than
- **>** : Greater than
- **≤** : Less than or equal to
- **≥** : Greater than or equal to
- **±** : Plus-minus (indicates both positive and negative)
- **∓** : Minus-plus (used in conjunction with ±)

### Algebra
- **√** : Square root (e.g., √9 = 3)
- **∛** : Cube root
- **∜** : Fourth root
- **^** : Exponentiation (e.g., 2^3 = 8; sometimes written as superscript)
- **!** : Factorial (e.g., 5! = 120; also used for negation in logic)
- **%** : Percent (e.g., 50% = 0.5)
- **∑** : Summation (sigma notation)
- **∏** : Product (pi notation)

### Geometry
- **∠** : Angle (though it can also denote argument of a complex number, but in geometry it is standard)
- **⊥** : Perpendicular
- **∥** : Parallel
- **△** : Triangle
- **□** : Square or rectangle (sometimes used for quadrilateral)
- **⊙** : Circle (sometimes used for circle with center)
- **°** : Degree (e.g., 90°)

### Set Theory
- **∈** : Element of
- **∉** : Not an element of
- **⊂** : Subset of (proper subset sometimes, but often used for subset)
- **⊆** : Subset or equal to
- **⊇** : Superset or equal to
- **∪** : Union
- **∩** : Intersection
- **∅** : Empty set
- **ℕ** : Set of natural numbers
- **ℤ** : Set of integers
- **ℚ** : Set of rational numbers
- **ℝ** : Set of real numbers
- **ℂ** : Set of complex numbers

### Logic
- **∧** : Logical AND (conjunction)
- **∨** : Logical OR (disjunction)
- **¬** : Logical NOT (negation)
- **⇒** : Implies (if...then)
- **⇔** : If and only if (equivalence)
- **∀** : For all (universal quantifier)
- **∃** : There exists (existential quantifier)
- **∄** : There does not exist
- **∴** : Therefore
- **∵** : Because

### Calculus and Analysis
- **∫** : Integral
- **∬** : Double integral
- **∭** : Triple integral
- **∮** : Contour integral
- **∇** : Del (gradient, divergence, curl)
- **∂** : Partial derivative
- **∞** : Infinity
- **lim** : Limit
- **Δ** : Change (e.g., Δx) or Laplacian
- **∑** : Summation (also in calculus)

### Number Theory
- **∣** : Divides (e.g., 3 ∣ 6)
- **∤** : Does not divide
- **≡** : Congruence (modular arithmetic)
- **⌊ ⌋** : Floor function
- **⌈ ⌉** : Ceiling function

### Probability and Statistics
- **μ** : Mean (population mean)
- **σ** : Standard deviation
- **σ²** : Variance
- **ρ** : Correlation coefficient
- **∩** : Intersection of events (also in sets)
- **∪** : Union of events
- **P(A)** : Probability of event A
- **E[X]** : Expected value
- **Var(X)** : Variance

### Miscellaneous
- **∼** : Similar to (geometry) or distributed as (statistics) or equivalence relation
- **≈** : Approximately equal
- **∝** : Proportional to
- **≡** : Identically equal (also congruence)
- **→** : Approaches (in limits) or mapping
- **↦** : Maps to (function definition)
- **°** : Degree (angle) or degree of a polynomial (rare)

### Note on Ambiguity
While these symbols are standard, some can have multiple meanings in different branches of mathematics. For example:
- **|** can mean absolute value, cardinality, or divisibility.
- **!** can mean factorial or logical negation.
- **×** can mean multiplication or Cartesian product.
- **∠** can mean angle or argument of a complex number.

The list above focuses on symbols that are most commonly used with a single primary meaning in typical mathematical contexts.
