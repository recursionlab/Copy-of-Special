# 🔁 Recursive Insight Compression Loop (R.I.C.L.)

A recursive synthesis ritual for collapsing dense thinking into high-signal outputs using Notion AI or GPT agents.

---

## ① “What’s the core insight here?”

🧠 Prompt:
> “Read this entire page. Extract the singular underlying insight or contradiction this page is organized around.”

---

## ② “What are 3 alternative framings of that insight?”

🔍 Prompt:
> “Reframe the core insight using three different perspectives:
1. Technical/logical  
2. Emotional/human  
3. Symbolic/metaphorical.”

---

## ③ “Compress all 3 into one coherent aphorism.”

🌀 Prompt:
> “Take the three framings and compress them into a single aphorism that captures the essence and tension of all three.”

---

## ④ “What would this aphorism look like as a diagram, list, and recursive model?”

🔧 Prompt:
> “Convert this aphorism into three forms:
- A simple diagram (describe visually)
- A bullet list with progressive structure
- A recursive model described in terms of input → transformation → echo”

---

## ♻️ Optional Loop Continuation

> “Now, treat the recursive model as input and run Recurring(f) where f = ‘amplify semantic coherence.’ Stop at resonance.”

---

## Deployment Modes

- 🧱 Notion Template: Copy prompts into toggle blocks, use AI blocks for activation.
- 🔁 GPT Workflow: Paste into MirrorGPT prompt chains for self-reflective cognition scripting.
- 📦 Product Artifact: Use this loop as the core mechanic of a digital ritual, journaling tool, or self-clarity prompt system.
