Framework expansion — 1000 practical ideas for a personal research-mining system
=================================================================================

Scope and method
----------------
This document synthesizes ideas tailored to a single-user research-mining setup, drawing on the `meta-knowledge` corpus filenames and the existing repository structure. Items are concise, actionable, and organized into themes so you can scan and pick what you want to implement next.

How to use this file
--------------------
- Scan theme headings for areas you care about (Extraction, Canonicalization, Clustering, MLOps, UX, Experiments, etc.).
- Pick one idea, implement it, and mark it as done in your own TODO tracker.
- Many items are deliberately small and composable: combine 3–5 to get meaningful improvements quickly.

Legend: items are short commands or features; effort depends on familiarity (tiny, small, medium).
